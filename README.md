## 📌 About the Project

This project recreates the look and feel of Spotify's landing/home page interface. It was built as a practice project to strengthen my understanding of responsive layouts, Bootstrap's grid system, and clean UI structuring using plain HTML and CSS — no JavaScript or backend involved.

## 🛠️ Tech Stack

- **HTML5** — page structure and semantic markup
- **CSS3** — custom styling, layout, and responsiveness
- **Bootstrap** — grid system, components, and responsive utilities

## ✨ Features

- Responsive layout that adapts to desktop, tablet, and mobile screens
- Spotify-inspired sidebar navigation
- Album/playlist card layout using Bootstrap grid
- Custom color theme matching Spotify's dark UI
- Hover effects and smooth transitions

## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/abhi7044/spotify-clone.git

# Navigate into the project folder
cd spotify-clone

# Open index.html in your browser
```

No build tools or dependencies required — just open `index.html` directly, or use a live server extension (e.g. VS Code Live Server) for auto-reload while editing.
